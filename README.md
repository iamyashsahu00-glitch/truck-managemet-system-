# 🚛 Truck Management System

A simple Truck Management System developed to manage truck trips, driver cash, and daily expenses efficiently.

## ✨ Features

- Manage truck and driver details
- Track opening and closing cash
- Record diesel, food, and maintenance expenses
- Manage driver advances
- Maintain trip records
- Store remarks and transaction history

## 🛠️ Technologies Used

- Python
- CSV/Excel
- Pandas
- OpenPyXL

## 🎯 Purpose

This project helps transport businesses maintain organized financial and trip records while reducing manual paperwork.
